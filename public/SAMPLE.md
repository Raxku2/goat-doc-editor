## What is Markdown?

Markdown is a **lightweight markup language** used to format text using simple symbols.

It is:

* Easy to read
* Easy to write
* Widely used (GitHub, documentation, blogs)

You write plain text ‚Üí Markdown converts it into formatted content.

---

## Basic Markdown Syntax

### Headings

```md
# Heading 1
## Heading 2
### Heading 3
```

Result:

# Heading 1

## Heading 2

### Heading 3

---

### Bold, Italic, Strikethrough

```md
**Bold text**
*Italic text*
~~Strikethrough~~
```

Result:

* **Bold text**
* *Italic text*
* ~~Strikethrough~~

---

### Lists

#### Unordered List

```md
- Item one
- Item two
  - Sub item
```

#### Ordered List

```md
1. First
2. Second
3. Third
```

---

### Links

```md
[GitHub](https://github.com)
```

Result: [GitHub](https://github.com)

---

### Images

```md
![Alt text](https://via.placeholder.com/150)
```

---

### Code

#### Inline Code

```md
Use `console.log()` to print output.
```

#### Code Block

```js
function hello() {
  console.log("Hello World");
}
```

---

### Blockquote

```md
> This is a blockquote.
```

Result:

> This is a blockquote.

---

### Horizontal Line

```md
---
```

## How to Use This Editor

### 1. Write Markdown

* Use the **editor panel** to type Markdown
* Formatting updates **live in the preview**

### 2. Live Preview

* The right panel shows rendered output
* Changes update instantly

### 3. Save Your Work

* Stored securely using **MongoDB**

### 4. Download / Export File

You can download your document as:

* `.md`
* `.txt`
* `.pdf` (if enabled)
